Multiplicação

A multiplicação faz parte das 4 operações básicas da matemática, sendo uma adição sucessiva de um número por ele mesmo

Representamos a multiplicação usando o símbolo "x" ou "."

O nome dado ao resultado da multiplicação é chamado de produto, e os números que estão sendo multiplicados são chamados de fatores

É importante conhecer a tabuada para se realizar os cálculos, para acessar a tabuada, Clique aqui

A multiplicação é usada para facilitar uma adição em sequência de um número por ele mesmo, veja o exmplo: Se quisermos somar 5 + 5 + 5 + 5 + 5 o resultado será 25. Agora se fizermos 5 X 5, obteremos o mesmo resultado, porém a conta é bem menor que a anterior

O símbolo de "x" é chamado de vezes, ou seja, na conta acima, estamos fazendo 5 vezes 5

Como realizamos o cálculo

Primeiramente, deve se organizar os números um sobre o outro, assim como se faz na soma e subtração, colocando o de maior número por cima:

Agora realizamos a multiplicação começando pela unidade, no caso, 5 vezes 0 = 0. Agora colocamos o resultado abaixo do 5:

Agora fazemos a multiplicação pela dezena, nesse caso, 5 x 1 = 5, e colocamos o resultado a esquerda do anterior:

Sendo assim, o produto de 10 vezes 5 é igual a 50

Propriedades da multiplicação

A multiplicação possui algumas propriedas, sendo elas

Propriedade comutativa:

Em uma multiplicação, a ordem dos fatores não altera o valor do produto: a x b = b x a,
2 x 3 = 3 x 2 = 6

Pripriedade distributiva:

envolve a adição e multiplicação: a.(b+c) = ab+ac
5(1 + 2) = 5+10 = 15

Pode se resolver desta maneira também, 5(1 + 2) = 5(3) = 15

Propriedade associativa

A associação entre os termos gerará o mesmo produto

Exemplo:(a x b) x c = (b x c ) x a
(5x2)x3 = (3 x 5)x 2 = 30

Elemento neutro

Na multiplicação, o 1 é elemento neutro, o que significa que qualquer número multiplicado por 1, é igual a ele mesmo

ax1 = a, 10x1 = 10

Regra de sinais

+ x + = +| - x - = +| + x - = -| - x + = -

Exemplo: - 10 x 1 = 10, -10 x (-1) = 10, 5 x (-2) = -10, -5 x 3 = -15
